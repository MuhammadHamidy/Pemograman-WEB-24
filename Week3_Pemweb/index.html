Pemograman Website

Pemograman Website merupakan mata kuliah program studi ilmu komputer yang membahas mengenai seputar website. dimulai dari pengenalan Http (Hypertext Transfer Protocol) dan Https (Hypertext Transfer Protocol Secure) yang berfungsi sebagai dasar komunikasi antar klien (browser) dan server, hingga komponen dasar pembuatan website (html, css dan javascript). Website menjadi salah satu media utama untuk menyampaikan informasi, layanan, hingga membangun komunikasi antar pengguna secara global. sehingga, kemampuan dalam membuat dan mengembangkan sangat penting khususnya di bidang ilmu komputer.

Adapun komponen dasar yang digunakan dalam pembuatan halaman website.

  1. HTML

    2. Html (Hypertext Markup language) merupakan bahasa markup yang digunakan untuk membuat halaman website. Html dapat digunakan untuk mendefinisikan struktur halaman website seperti elemen paragraf, gambar, link dan lain sebagainya. Sebagian besar halaman website yang berisikan elemen HTML dan didalamnya terletak pula css dan Javascript

  2. CSS

    3. Css (Cascading Style Sheets) merupakan bahasa untuk mengatur ampilah (gaya) pada elementa HTML. atau dapat diartikan sebagai bahasa yang mendesain dari elemen HTML. pada Css kita dapat melakukan styling elemen HTML seperti background, border, posisi, dan lain sebagainya.

  3. JavaScript

    4. JavaScript merupakan sebuah bahasa pemograman yang digunakan untuk membuat website lebih interaktif. sehingga, kita dapat membuat fitur - fitur pada halaman website seperti menampilkan elemen tertentu hanya pada browserr tertentu, serta dapat menyesuaikan font pada web, dan lain sebagainya.
